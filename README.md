## Local Development Setup
1. Clone the project
    ```
    git clone https://github.com/masif2002/quizzy
    ```
2. Install dependencies
    ```
    flutter pub get
    ```
3. Launch emulator (or Connect your device)
    ```
    flutter emulator --launch <your_emulator_id>
    ```
3. Run the flutter app
    ```
    flutter run
    ```
